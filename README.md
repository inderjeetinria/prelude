## Prerequisites

```bash
$ brew update
$ brew install git
$ brew install ctags
$ brew install emacs --use-git-head --cocoa --srgb
```

```bash
$ ln -s /usr/local/Cellar/emacs/24.3/Emacs.app /Applications
```
